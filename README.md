# Myntra-Fashion-Clothing-Analysis-Sales-Discounts-and-Customer-Behavior

# Introduction:

This project analyzes fashion product data to understand sales trends, discounts, and customer behavior. It helps identify popular categories, top brands, pricing strategies, and customer preferences.

# Objective:

The objective of this project is to analyze product distribution, brand performance, customer ratings, size availability, and pricing-discount patterns in order to:

•	Identify popular categories and gender-wise distribution

• Analyze top-performing brands

• Understand customer ratings and satisfaction

• Study size availability and inclusivity

• Evaluate pricing and discount patterns

# Data description:

• Product-level data (4,800+ records)

•	This dataset contains product-level details from a fashion retail store.

•	Each row represents one product, showing its ID, brand, category, gender target, price, discount, final price, size options, and customer feedback (ratings and reviews).

# Tools Used:

• Excel

• Power Query

• Pivot Tables

# Data Cleaning (Excel/Power Query):

• Removed duplicates and handled missing values

• Standardized text fields (brand, category, sizes)

• Converted prices and ratings into proper formats

• Verified: Final Price = Original Price – Discount

# Key Analysis:

• Price ranges from ₹185 to ₹9,499 (wide variation)

• Average product price ≈ ₹1,600

• Discounts vary widely, with some very high discount offers

• Data shows skewed distribution (few high-price items increase average)

# Key Insights:

•	Women’s category dominates the product catalog

• Western wear is the top category

• Roadster is the leading brand by product count

• Customer ratings are high (~4.5+), showing strong satisfaction

• Most products are available in standard sizes (S–XL/XXL)

• Higher-priced products tend to have larger discounts
